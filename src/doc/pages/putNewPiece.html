駒を任意のマスに配置する

ボードの任意のマスに駒を配置する。
Board.putNewPiece()を用いて実行。

メソッド仕様

Board.prototype.putNewPiece(piece, pX, pY, playeaIdOrDeg, {displayPtn, isMoved}): void
  • piece: string
    • 駒の表現文字
  • pX: number
    • X方向配置位置(マス目基準)
  • pY: number
    • Y方向配置位置(マス目基準)
  • playeaIdOrDeg: number
    • プレイヤー番号または駒の配置角
  • displayPtn: number
    • 表示文字列を変更(1〜)
  • isMoved: boolean
    • 初回移動済みか否か
