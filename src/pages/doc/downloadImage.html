スクリーンショットを保存する

ボード情報を画像として保存する。
デフォルトの操作パネルの表示で提供される機能の一つであるが、
独自のボタン表示や実行タイミングの制御を行える。
Board.downloadImage()を用いて実行。

メソッド仕様

Board.prototype.downloadImage(fileName, ext, urlType): Promise<void>
  • fileName: string = "shogicross"
    • ファイル名
  • ext: string = "png"
    • 保存したい画像形式または拡張子
  • urlType: "base64"|"blob" = "base64"
    • 処理するデータ形式、"blob"の方が処理が軽くなるが互換性に劣る
