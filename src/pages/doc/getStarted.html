CDNから使用する

iife版

html中にscriptタグを埋め込んで使用する場合、 
<script src="https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.6/src/dist/ShogiCross.iife.js"></script>
または
<script src="https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.6/src/dist/ShogiCross.iife.min.js"></script>
<script>
    const {Board, gameSoft} = ShogiCross;
    const board = Board.run(canvas, gameSoft.shogi);
</script>

ES Module版

ES Moduleでimportして使用する場合、 
<script type="module">
    import {Board, gameSoft} from "https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.6/src/dist/ShogiCross.js";
    または
    import {Board, gameSoft} from "https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.6/src/dist/ShogiCross.min.js";

    const board = Board.run(canvas, gameSoft.shogi);
</script>

CPF対応版

CodePen等CPF(セキュリティポリシー)の問題で外部ファイル読込みに制限がある場合、代わりに下記を指定する。 
<!--iife版-->
<script src="https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.6/src/dist/ShogiCross_nojson.iife.js"></script>
<script src="https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.6/src/dist/ShogiCross_nojson.iife.min.js"></script>

/* ES Module版 */
import {Board, gameSoft} from "https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.6/src/dist/ShogiCross_nojson.js";
import {Board, gameSoft} from "https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.6/src/dist/ShogiCross_nojson.min.js";
