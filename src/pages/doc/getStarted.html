CDNから使用する

iife版

html中にscriptタグを埋め込んで使用する場合、 
<script src="https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.17/src/dist/ShogiCross.iife.js"></script>
または
<script src="https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.17/src/dist/ShogiCross.iife.min.js"></script>
<script>
    const {Board, gameSoft} = ShogiCross;
    const board = Board.run(canvas, gameSoft.shogi);
</script>

ES Module版

ES Moduleでimportして使用する場合、 
<script type="module">
    import {Board, gameSoft} from "https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.17/ShogiCross.js";
    または
    import {Board, gameSoft} from "https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.17/ShogiCross.min.js";

    const board = Board.run(canvas, gameSoft.shogi);
</script>

ゲームデータ拡張非対応版

ゲームデータ拡張(外部ファイル読込み)が不要な場合に使用しても良い。
余計な通信が発生しないので開発者コンソールが汚れないし、ちょっとだけ読み込みが早くなる…かも。 
<!--iife版-->
<script src="https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.17/src/dist/ShogiCross_nojson.iife.js"></script>
<script src="https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.17/src/dist/ShogiCross_nojson.iife.min.js"></script>

/* ES Module版 */
import {Board, gameSoft} from "https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.17/src/dist/ShogiCross_nojson.js";
import {Board, gameSoft} from "https://cdn.jsdelivr.net/gh/yosgspec/ShogiCross@0.17/src/dist/ShogiCross_nojson.min.js";

ゲームデータ拡張

ボードやマス目についてゲームデータを拡張を行える。
json形式のデータを/json/ShogiCross/以下に配置する。
サンプルデータはダウンロード(サイト、GitHub)またはnpmで用意できる。
配置パスは読み出すhtmlファイルのルートディレクトリとなる。
