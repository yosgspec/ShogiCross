# pieceCost.json
駒の価値をリスト化したデータ。  
駒の一文字をキーとした辞書。  
(駒の一文字表記 = pieces.jsonに列挙されるキー)

## 駒の価値 {number}
駒の価値は整数で記載。  
ルールとしておおよそ下記を定めている。

* 捕縛属性を持つ駒は持たない駒と比較して1.5倍程度の価値となる。
* 王駒は不利であるほど負数が増大する。
* 実状として、盤の大きさによって走り駒の価値は変動するが固定値とする。

```json
"仕":  4,
"卆":  4,
"歩":  3,
"兵":  3,
"ひ":  2,
"燕":  2,
"貝":  2,
"卒":  2,
"央":  2,
```
